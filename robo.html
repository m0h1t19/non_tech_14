This Year's Summer Projects

Project 1: Gyrobot : self-balancing robot - A robot designed to move and balance itself on its own. It will be able to keep track of its movement and balance itself .

Project 2: Space Station : controls and communication - A part of mars rover project . In this project we will be learning to design an overall control system to communicate to the robot over large distances.

Project 3: Autonomous navigation of mars rover - The idea of this project is to work on the navigation and path planning of the mars rover.

Project 4: Robotic arm - The idea of this project is to build robotic arm which is capable of grabbing and lifting things.
